---
title: 'Get HIP-4 Outcome Holders'
openapi: openapi-hyperliquid.yaml GET /hyperliquid/outcomes/{outcomeId}/holders
description: Top holders of a HIP-4 outcome with classification badges
---

Returns the top holders of a HIP-4 outcome, ranked by net position size.
Each holder includes classification badges (`isWhale`, `isInsider`, `isSmartMoney`)
plus a composite `smartScore`.

Rate limit: 60 requests/minute.

## Path Parameters

<ParamField path="outcomeId" type="integer" required>
  HIP-4 outcome ID (numeric)
</ParamField>

## Query Parameters

<ParamField query="side" type="string" default="both">
  Filter by side: `both`, `yes`, `no`
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Page size (max 100)
</ParamField>

## Examples

<CodeGroup>
```bash cURL
curl "https://api.klarity.trade/hyperliquid/outcomes/123/holders?side=yes&limit=25"
```

```typescript TypeScript
const res = await fetch(
  'https://api.klarity.trade/hyperliquid/outcomes/123/holders?side=yes&limit=25',
);
const { data, meta } = await res.json();
// data[0]: { address, side, netSize, notional, tradeCount, lastFillTs,
//            isWhale, isInsider, isSmartMoney, smartScore }
```
</CodeGroup>
