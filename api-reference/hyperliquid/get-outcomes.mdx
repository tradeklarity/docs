title: 'List HIP-4 Outcomes'
openapi: openapi-hyperliquid.yaml GET /hyperliquid/outcomes
description: Paginated list of Hyperliquid HIP-4 outcomes with mid prices and 24h volume
---

Returns Hyperliquid HIP-4 outcomes (tradeable binary contracts) with full metadata, current mid prices (yesMidPx/noMidPx), and 24h volume.

**Terminology:**
- **Outcome** = tradeable binary (≈ Polymarket market)
- **Question** = container grouping outcomes (≈ Polymarket event)
- **Standalone outcome** = outcome with no question wrapper (HIP-4-specific)

For a unified discovery experience with questions and embedded outcomes, use `POST /hyperliquid/questions/discover`. For outcome-level drilling, use this endpoint or `GET /hyperliquid/outcomes/{outcomeId}`.
