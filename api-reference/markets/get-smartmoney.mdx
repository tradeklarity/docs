## Overview

Returns holders classified as **smart money** for a specific market. Smart money wallets have a proven track record of profitable trading across Polymarket.

<Card title="Holder Classification" icon="lightbulb" href="/concepts/holder-classification">
  Learn how whale, smart money, and insider classifications work.
</Card>

## Path Parameters

<ParamField path="slug" type="string" required>
  Market slug identifier (e.g., `will-trump-win-2024`)
</ParamField>

## Smart Money Criteria

All three conditions must be met:

| Criteria         | Threshold | Description                                                       |
| ---------------- | --------- | ----------------------------------------------------------------- |
| Win rate         | >= 60%    | Percentage of closed positions that were profitable               |
| Realized PnL     | >= $5,000 | Total realized profit across all closed positions                 |
| Closed positions | >= 20     | Minimum number of resolved positions for statistical significance |
