## Common Patterns

### Pagination

Most list endpoints support `limit` and `offset`:

| Parameter | Type    | Default | Description             |
| --------- | ------- | ------- | ----------------------- |
| `limit`   | integer | 25      | Maximum items to return |
| `offset`  | integer | 0       | Items to skip           |

### Error Format

```json
{
  "error": "Error message",
  "details": [],
  "traceId": "abc-123-def"
}
```

See [Error Codes](/api-reference/error-codes) and [Rate Limits](/api-reference/rate-limits) for details.
