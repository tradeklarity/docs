---
title: Insider Tracker Stream (Public)
openapi: openapi-stream.yaml GET /insider-tracker/public/stream
description: SSE stream of insider trade alerts with fixed thresholds (no auth)
---

Fixed thresholds: $10k notional min, 5 markets max, no account age filter.

## SSE Events

### `event: system`

```json
{ "type": "connected", "config": { "notionalMin": 10000, "uniqueMarketsMax": 5 } }
```
