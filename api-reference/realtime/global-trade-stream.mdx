---
title: Global Trade Stream
openapi: openapi-stream.yaml GET /trades/stream
description: SSE stream of all Polymarket trades above $100 notional (no auth)
---

Streams all trades above $100 notional, enriched with trader stats and classification flags. Up to 50 recent trades sent on connect as backfill.

## SSE Events

### `event: system`

```json
{ "type": "connected", "config": { "notionalMin": 100 } }
```

### `event: trade`

Same `TradeAlertEvent` schema as all other streams. Both backfill and live trades use this event type.
