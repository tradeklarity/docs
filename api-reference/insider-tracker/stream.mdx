---
title: Insider Tracker Stream (Authenticated)
openapi: openapi-stream.yaml GET /insider-tracker/stream
description: SSE stream of insider trade alerts with per-user configurable thresholds
---

## Headers

<ParamField header="x-user-id" type="string">
  User identifier (defaults to "anonymous")
</ParamField>

## Query Parameters

<ParamField query="since" type="string">
  ISO 8601 timestamp. Historical trades since this time are sent as `event: trade` before live streaming begins.
</ParamField>

## SSE Events

All trades (live and backfill) use the same `event: trade` format with the unified `TradeAlertEvent` schema.

### `event: system`

```json
{ "type": "connected", "config": { "notionalMin": 10000, "uniqueMarketsMax": 5, "maxAccountAgeHours": null } }
```

Config updates while connected:

```json
{ "type": "config_updated", "config": { "notionalMin": 5000, "uniqueMarketsMax": 10, "maxAccountAgeHours": 24 } }
```
