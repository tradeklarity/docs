---

## Overview

Returns paginated historical trades for a market with full trader classifications (whale, smart money, insider) and optional profile data.

**Use this endpoint for:**

- Initial data load / backfill before connecting to real-time stream
- Fetching historical trades after a disconnect
- Populating trade history UI

<Card title="Real-Time Stream" icon="bolt" href="/api-reference/markets/stream">
  For live trade updates, use `GET /markets/{slug}/stream` (SSE).
</Card>

## Path Parameters

<ParamField path="slug" type="string" required>
  Market slug identifier (e.g., `russia-x-ukraine-ceasefire-in-2025`)
</ParamField>

## Query Parameters

<ParamField query="limit" type="integer" default="25">
  Maximum number of trades to return (max: 100)
</ParamField>

<ParamField query="offset" type="integer" default="0">
  Number of trades to skip for pagination
</ParamField>

<ParamField query="includeProfiles" type="boolean" default="false">
  Include Polymarket profile data (displayName, profileImage, xUsername, etc). Set to `false` for
  faster responses (~100ms), `true` to include profiles (~500ms+).
</ParamField>

## Response

  <Expandable title="Trade object">

    </ResponseField>

      Market slug
