## Overview

Streams real-time trading signals from tracked top traders via Server-Sent Events. Each signal is an enriched trade event containing the full trade context plus the trader's leaderboard stats. Supports filtering by Sharpe ratio, win rate, PnL, and trader tier.

<Info>
Legacy path `/copy-trading/stream` works identically and is maintained for backwards compatibility.
</Info>

## Query Parameters

<ParamField query="minSharpe" type="number">
  Minimum Sharpe ratio. Only signals from traders with `sharpeRatio >= minSharpe` are sent.
</ParamField>

<ParamField query="minWinRate" type="number">
  Minimum win rate (0-1). Only signals from traders with `winRate >= minWinRate` are sent.
</ParamField>

<ParamField query="minPnl" type="number">
  Minimum total PnL in USD. Only signals from traders with `totalPnl >= minPnl` are sent.
</ParamField>

<ParamField query="traderTier" type="string" default="all">
  Filter by trader tier: `elite` (top 10), `top` (top 100), or `all` (no filter).
</ParamField>

<ParamField query="since" type="string">
  ISO 8601 timestamp for historical backfill. If provided, the stream replays persisted signals from ClickHouse before switching to live streaming.
</ParamField>

## SSE Events

### `system` — Connection confirmation

Sent immediately on connect. Contains the filter that was applied.

```json
{
  "type": "connected",
  "filter": { "minSharpe": 1.5, "traderTier": "elite" }
}
```

### `backfill` — Historical replay

Sent once after connect if `since` was provided. Contains an array of signal objects.

### `signal` — Live trading signal

Each signal is a JSON object with trade details and trader leaderboard context:

<RequestExample>
```bash cURL
curl "https://ws.klarity.trade/stream/top-trader-signals/stream?minSharpe=1.5&traderTier=elite&since=2026-03-20T00:00:00Z" \
  --header "Accept: text/event-stream"
```

```javascript JavaScript
const eventSource = new EventSource(
  'https://ws.klarity.trade/stream/top-trader-signals/stream?minSharpe=1.5&traderTier=elite'
);

eventSource.addEventListener('signal', (e) => {
  const signal = JSON.parse(e.data);
  console.log(`[${signal.traderTier}] ${signal.wallet} ${signal.side} on ${signal.marketSlug} — $${signal.notionalUsd}`);
});

eventSource.addEventListener('backfill', (e) => {
  const signals = JSON.parse(e.data);
  console.log(`Received ${signals.length} historical signals`);
});

eventSource.addEventListener('system', (e) => {
  const msg = JSON.parse(e.data);
  console.log('Connected with filter:', msg.filter);
});

eventSource.onerror = (error) => {
  console.error('Stream error:', error);
  eventSource.close();
};
```
</RequestExample>

## Connection Limits

- **Per-IP limit**: 10 concurrent SSE connections
- **Rate limit**: 429 Too Many Requests if exceeded
- **Heartbeat**: `: heartbeat` comment sent every 15 seconds
- **Reconnection**: Exponential backoff recommended (start at 1s, max 60s)
