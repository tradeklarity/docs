---
title: Smart Money Stream (Public)
openapi: openapi-stream.yaml GET /smart-money/public/stream
description: SSE stream of trades from proven profitable traders (no auth)
---

Fixed thresholds: $50k+ realized PnL, 60%+ win rate, 10+ markets traded, $5k notional min.

## Query Parameters

<ParamField query="since" type="string">
  ISO 8601 timestamp. Historical trades since this time are sent as `event: trade` before live streaming begins.
</ParamField>

## SSE Events

### `event: system`

```json
{ "type": "connected", "config": { "minRealizedPnlUsd": 50000, "minWinRate": 0.6, "minMarketsTraded": 10 } }
```
