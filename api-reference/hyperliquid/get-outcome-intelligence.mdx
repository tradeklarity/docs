---
title: 'Get HIP-4 Outcome Intelligence'
openapi: openapi-hyperliquid.yaml GET /hyperliquid/outcomes/{outcomeId}/intelligence
description: AI-generated summary for a Hyperliquid HIP-4 outcome
---

Latest LLM-generated intelligence summary for a single HIP-4 outcome.
Pre-generated by the `generate-hl-outcome-intelligence` sync job every
30 minutes for the top active outcomes by 24h volume.

Mirrors the Polymarket
[`GET /markets/:slug/intelligence`](/api-reference/markets/get-intelligence)
endpoint but keyed on HL `outcomeId`.

### Key fields

- **summary** — 1-3 sentence data-grounded analysis of the outcome's
  current state, momentum, and notable signals.
- **headline** — short (<= 80 char) headline suitable for cards.
- **confidence** — model self-reported confidence (0-1).
- **tokensUsed** — approximate OpenRouter token spend on the run that
  produced this summary.
- **model** — OpenRouter model identifier (defaults to Gemini Flash Lite).
- **generatedAt** — ISO-8601 timestamp of generation.

## Path Parameters

<ParamField path="outcomeId" type="integer" required>
  HIP-4 outcome ID (numeric)
</ParamField>

## Examples

<CodeGroup>
```bash cURL
curl "https://api.klarity.trade/hyperliquid/outcomes/123/intelligence"
```

```typescript TypeScript
const res = await fetch(
  'https://api.klarity.trade/hyperliquid/outcomes/123/intelligence',
);
const { data } = await res.json();
// data.summary, data.headline, data.confidence
```
</CodeGroup>

<Note>
  Returns 404 when no summary exists for the outcome yet — the sync job only
  summarizes the top N outcomes by 24h volume.
</Note>
