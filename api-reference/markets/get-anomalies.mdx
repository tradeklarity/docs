---
title: 'Anomaly Detection'
openapi: 'GET /markets/anomalies'
description: 'Detect unusual market activity including volume spikes, holder shifts, and price divergences'
---

Retrieve detected anomalies across all prediction markets. Anomalies are automatically identified patterns that deviate from normal market behavior, such as sudden volume spikes, rapid holder count changes, or unexpected price movements.

Use this endpoint for surveillance dashboards, alert systems, or to find markets with unusual activity that may warrant closer investigation.

### Key Features

- **Three anomaly types** -- volume spikes, holder shifts, and price divergences
- **Severity levels** -- low, medium, high, and critical classifications
- **Sortable results** -- sort by detection time or severity
- **Paginated** -- supports offset-based pagination for large result sets

### Query Parameters

| Parameter | Type | Default | Description |
| --- | --- | --- | --- |
| `limit` | integer | 20 | Results per page (1-100) |
| `offset` | integer | 0 | Pagination offset |
| `anomaly_type` | string | *all* | Filter by type: `volume_spike`, `holder_shift`, or `price_divergence` |
| `severity` | string | *all* | Filter by severity: `low`, `medium`, `high`, or `critical` |
| `sort_by` | string | `detected_at` | Sort field: `detected_at` or `severity` |
| `sort_order` | string | `DESC` | Sort direction: `ASC` or `DESC` |

### Example Requests

**Get all recent anomalies:**

```bash
curl "https://api.klarity.trade/markets/anomalies"
```

**Get critical volume spikes only:**

```bash
curl "https://api.klarity.trade/markets/anomalies?anomaly_type=volume_spike&severity=critical"
```

**Get anomalies sorted by severity:**

```bash
curl "https://api.klarity.trade/markets/anomalies?sort_by=severity&sort_order=DESC&limit=10"
```

### Response Format

| Field | Type | Description |
| --- | --- | --- |
| `data` | array | Array of anomaly objects |
| `data[].anomaly_id` | string | Unique anomaly identifier |
| `data[].market_slug` | string | Affected market slug |
| `data[].anomaly_type` | string | Type of anomaly detected |
| `data[].severity` | string | Severity level |
| `data[].description` | string | Human-readable description of the anomaly |
| `data[].data_context` | string | Additional context data (JSON string) |
| `data[].detected_at` | string | ISO-8601 timestamp of detection |
| `count` | number | Number of results in this page |
| `offset` | number | Current pagination offset |

### Anomaly Types

| Type | Description |
| --- | --- |
| `volume_spike` | Trading volume significantly exceeds the market's historical norm |
| `holder_shift` | Rapid increase or decrease in the number of holders |
| `price_divergence` | Price movement inconsistent with holder sentiment or volume |
