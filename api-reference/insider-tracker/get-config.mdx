---
title: Get Insider Config
openapi: openapi-stream.yaml GET /insider-tracker/config
description: Get your current insider tracker configuration and server constraints
---

## Overview

Retrieve your current insider tracker configuration along with the server-enforced constraints. If you haven't set a custom config, the default values are returned.

## Headers

<ParamField header="x-user-id" type="string" required>
  User identifier
</ParamField>

## Response

<ResponseField name="config" type="object">
  Your current configuration

  <Expandable title="config properties">
    <ResponseField name="notionalMin" type="number">
      Minimum trade notional value in USD to trigger an alert.

      Default: `10000`
    </ResponseField>

    <ResponseField name="uniqueMarketsMax" type="number">
      Maximum number of unique markets the trader can have participated in.

      Default: `5`
    </ResponseField>

    <ResponseField name="maxAccountAgeHours" type="number | null">
      Maximum account age in hours. Set to `null` to disable this filter.

      Default: `null` (disabled)
    </ResponseField>

  </Expandable>
</ResponseField>

<ResponseField name="constraints" type="object">
  Server-enforced floor and ceiling values

  <Expandable title="constraints properties">
    <ResponseField name="notionalMin.floor" type="number">
      Minimum allowed value for `notionalMin`: `1000`
    </ResponseField>

    <ResponseField name="uniqueMarketsMax.ceiling" type="number">
      Maximum allowed value for `uniqueMarketsMax`: `20`
    </ResponseField>

    <ResponseField name="maxAccountAgeHours.ceiling" type="number">
      Maximum allowed value for `maxAccountAgeHours`: `48`
    </ResponseField>

  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl "https://ws.klarity.trade/stream/insider-tracker/config" \
  -H "x-user-id: my-user-id"
```

```javascript JavaScript
const response = await fetch('https://ws.klarity.trade/stream/insider-tracker/config', {
  headers: { 'x-user-id': 'my-user-id' },
});
const data = await response.json();
console.log('Config:', data.config);
console.log('Constraints:', data.constraints);
```

</RequestExample>

<ResponseExample>
```json
{
  "config": {
    "notionalMin": 10000,
    "uniqueMarketsMax": 5,
    "maxAccountAgeHours": null
  },
  "constraints": {
    "notionalMin": { "floor": 1000 },
    "uniqueMarketsMax": { "ceiling": 20 },
    "maxAccountAgeHours": { "ceiling": 48 }
  }
}
```
</ResponseExample>

## Default vs Custom Config

| Parameter            | Default         | You can set           |
| -------------------- | --------------- | --------------------- |
| `notionalMin`        | $10,000         | >= $1,000             |
| `uniqueMarketsMax`   | 5               | 1 - 20                |
| `maxAccountAgeHours` | null (disabled) | 1 - 48 hours, or null |
