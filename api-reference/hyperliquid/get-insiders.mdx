---
title: 'Get HIP-4 Outcome Insiders'
openapi: openapi-hyperliquid.yaml GET /hyperliquid/outcomes/{outcomeId}/insiders
description: Insider-flagged holders of a HIP-4 outcome
---

Returns addresses currently holding a HIP-4 outcome that meet the insider criteria:

- `account_age_hours <= INSIDER_MAX_AGE_HOURS`, **and**
- open notional `>= INSIDER_MIN_POSITION_USD`

For resolved outcomes, each row includes a `winningSide` boolean indicating
whether the holder's side matched the resolved winning side (null when active
or winning side is not yet reconciled).

Rate limit: 60 requests/minute.

## Path Parameters

<ParamField path="outcomeId" type="integer" required>
  HIP-4 outcome ID (numeric)
</ParamField>

## Query Parameters

<ParamField query="limit" type="integer" default="20">
  Max insiders to return (1-100)
</ParamField>

## Examples

<CodeGroup>

curl "https://api.klarity.trade/hyperliquid/outcomes/123/insiders?limit=20"
